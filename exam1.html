  1. First I need to compute c, so Iâ€™ll find the upper bound of \(\frac{f(t)}{g(t)}\)

\(\frac{f(t)}{g(t)} = \frac{(\frac{1}{3} + \frac{1}{8} x^2)}{\frac{1}{2}} = \frac{2}{3} + \frac{1}{4} x^2\)

  1. Stratified MC
strat_mc <- function(total_num, num_strat) {
  xjs <- seq(0, sqrt(2), length.out = num_strat + 1)
  theta = 0
  
  for (j in 1:num_strat) {
    X <- runif(total_num/num_strat, min=xjs[j], max=xjs[j+1])
    gx <- (2/sqrt(pi)) * exp(-X^2)
    theta = theta + (sqrt(2)/num_strat) * mean(gx)
  }
  return (theta)
}

strat_mc(1000,4)
## [1] 0.9557538
